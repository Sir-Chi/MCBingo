# Bingo

MC Bingo is a vanilla Minecraft survival scavenger hunt game mode, in which players need to explore, slay, mine and craft in order to fill out a Bingo card.

## How To Install

1) First download the datapack and then unzip the file.

2) To play the game, start by beginning to create a new minecraft world.

3) On the world creation screen, select the tab at the top of the screen called "More", then click the button called "Data Packs".

4) In this menu, click "Open Pack Folder" and copy and paste into this the unzipped "MC Bingo" folder.

5) Back in the "Selecting Data Packs" menu, hover over the icon for MC Bingo, this shows an arrow. Click this to select the datapack to be used when the world is created. Then click "Done".

6) Make sure the world has "Allow Cheats" enabled.

7) Once you load into the world, run the command "/function bingo:master/setup" directly, or run the command "/reload" to have the command appear in chat for you to click. This will create the Bingo Starting Lobby.

8) If on a server, make sure that command blocks are turned on in the server configs.

## Game Features

There is a pool of 116 items that can appear on the Bingo Card.

**Teams**  
Any number of players are supported. There are 4 team colours that can be chosen from. There is an option within the Lobby to randomise or clear teams. 

**Seeds**  
Bingo features "Seeds" where you can input a number code and it will create a Bingo card from that. The same "Seed" number always yields the same Bingo Card and team spawn locations. If using the same minecraft world seed, you can also share these codes with others so you can race using the same Card with the same team starting locations!

 **Custom World**  
Bingo is played in a customised world, with 3 main changes:  

- Small Biomes - The overworld biomes have been customised and made much smaller.
- More Dungeons - Dungeon frequency is increased so that all players have a good chance of finding a dungeon whilst caving.  
- Stone Drops - Granite, diorite and andesite drops have been altered so that they drop cobblestone instead. This is to help with inventory management and to reduce clutter.  

## Gameplay

**Teams**  
Each team will get a set separate spawning location dependent on the Bingo card "Seed". At the start teams will get a quick overview of their spawn location before the game commences.

**The Bingo Card**  
Players are given a "Bingo Card" map picturing 25 different items. The goal is to race to collect these items as fast as possible to get 5-in-a-row Bingo.  

Each time you obtain an item on the card, a sound will play, your score will update and a text notification will appear in the chat.

You can see which Bingo items you have so far collected by "updating your map". To do so hold your maps and then drop one of them onto the ground.

**The Possible Items** 
All the possible items that can appear on the Bingo card can be viewed on a special advancement screen. This can be accessed in game by pressing "L" by default.

## Game Options

All of the game settings can be changed within the Bingo lobby which spawns in when you run the setup function.

**Game Modes**  
Bingo has a variety of different game modes. During the setup period of the game you can turn these game modes on or off.

All of these modes (except for Lockout and PvP) are active by default. So when the conditions for each mode are met an announcement is made  in the chat and you will be offered the  chance to end the game there or continue playing.

- 5-In-A-Line - The most basic Bingo mode. To win get 5 items in a row, column or diagonal.  
- Blackout - For extra challenge there is "Blackout" mode, where you need to get all 25 items on the Bingo card.  
- Timed - Gather as many items as possible within 25 minutes (1500 seconds) as a timed challenge.  
- Lockout - Once 1 team gets an item square on the Bingo card, no other team can get that square. First to Bingo or to get the high score wins the game. The high score depends on how many teams are active.  
- PvP - When active all teams will have the same spawn point. This means that they will all have to compete for the same resources.  

**Auto Ending**  
If you want the game to automatically end when the conditions for a certain game mode are met then you can turn that on in the config menu found in the lobby. Otherwise the game will play as normal. By default this if off.  

**Game Configs**  
During the setup period of the game, you can also customise the gameplay, for the duration of the game, with various pre-set configs:  

- Night Vision - Players get the Night Vision effect.  
- Speed - Players get the Speed II effect.  
- Depth Strider - Players get boots with Depth Strider III enchant.  
- Elytra - Players get an Elytra & Firework Rockets.  
- All Recipes - Players get all recipes unlocked in the recipe book.  
- Condensed Teams - This makes the scoreboard information more concise by using only the team names instead of all the individual players. This is beneficial when there are large numbers or players.
